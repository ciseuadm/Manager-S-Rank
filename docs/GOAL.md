# 🎯 GOAL — S-RANK MANAGER BOT

> Этот документ — «компас» проекта. Перед любым нетривиальным действием сверяйся с ним.
> Если задача противоречит целям отсюда — стоп, сначала обнови GOAL.

## 1. Что мы строим

Telegram-бот в сеттинге аниме **Solo Leveling**, который:

1. **Наводит порядок в чатах** — автомодерация 24/7, наказание нарушителей,
   гибкие настройки под каждый чат.
2. **Геймифицирует общение** — ранги охотников (E → SSS), внутренняя валюта
   **Мана-руда**, топы, ежедневные бонусы, достижения.
3. **Растёт вирусно** — персональные реферальные ссылки, награды за приглашения,
   условия повышения (пригласи N → модератор/админ).
4. **Приносит доход владельцу** — рекламная сеть (1 пост в день во все чаты),
   реклама в канале бота, продажа Мана-руды и рекламных мест за Telegram Stars,
   обмен Мана-руды на подарки/Premium.

## 2. Легенда (tone of voice)

- Бот — **сильнейший охотник S-ранга**, «лучший дозорный».
- **Владелец (ты)** — **Монарх / Бог Системы**, который дал боту силу.
  Владелец стоит над всеми чатами, является супер-админом везде.
- Все тексты, ранги, валюта и события выдержаны в стиле Solo Leveling
  («Системное уведомление», «подземелье», «Мана-руда», «гильдия», «монарх теней»).

## 3. Для кого

- **Админы чатов/каналов** — получают бесплатный мощный модератор + инструменты
  роста сообщества (рефералки, повышения, статистика).
- **Участники чатов** — играют: качают ранг, фармят Мана-руду, зовут друзей.
- **Рекламодатели** — покупают показы в сети чатов и в канале бота.
- **Владелец** — управляет всем из единой панели, монетизирует аудиторию.

## 4. Ключевые требования (Definition of Done для проекта)

- [ ] Бот стабильно работает 24/7, переживает рестарты без потери данных.
- [ ] Масштабируется до тысяч чатов (БД, рассылки с rate-limit, очереди).
- [ ] Автомодерация настраивается на чат и не трогает админов/владельца.
- [ ] Полная экономика Мана-руды: заработок, кошелёк, магазин, обмен.
- [ ] Реферальная система с персональными ссылками и порогами повышения.
- [ ] Рекламная подсистема: кампании, расписание (1/день), таргет, статистика.
- [ ] Платежи через Telegram Stars (XTR) для покупки руды и рекламы.
- [ ] Удобная панель владельца для управления всем без правки кода.

## 5. Стек

- **Python 3.11+**, **aiogram 3.x** (long polling сейчас, webhook — при росте).
- **БД**: SQLite (старт) → миграционный слой → PostgreSQL (при масштабе).
- **Планировщик**: APScheduler (ежедневные рекламные рассылки, разбан по таймеру).
- **Платежи**: Telegram Stars (XTR) — нативно, без внешних эквайрингов.
- **Хостинг**: Railway / Fly.io (Docker), persistent volume под БД.
- **Логирование**: loguru.

## 6. Принципы и ограничения («нельзя»)

- ❌ Не спамить: реклама — максимум **1 пост в день на чат**, с возможностью
  для админа чата ограничить/отключить (но это влияет на условия бесплатности).
- ❌ Не хранить секреты в коде/гите (.env, токены, ключи).
- ❌ Не блокировать админов и владельца модерацией.
- ❌ Не обещать вывод реальных денег пользователям — Мана-руда меняется только
  на цифровые блага (подарки/Premium/реклама внутри бота).
- ✅ Любая фича сначала переиспользует существующий код, потом добавляет новый.
- ✅ Соблюдать лимиты Telegram (≈30 сообщений/сек на бота, троттлинг рассылок).
- ✅ Соответствовать правилам Telegram по рекламе и монетизации.
