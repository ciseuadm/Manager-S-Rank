# ⚔️ FEATURES — детализация функций

> Что бот уже умеет и что добавляем. Формат: ✅ готово / 🔧 доработать / 🆕 новое.

## 1. Модерация (ядро)

- ✅ Авто-удаление: NSFW, оскорбления, политика, спам, инвайт-ссылки.
- ✅ Антифлуд с авто-мутом.
- ✅ Опц. фильтры: ссылки, стикеры, CAPS.
- ✅ Эскалация: варн → мут → бан (лимиты на чат).
- ✅ Чёрный список слов на чат.
- ✅ Ручные команды: `/warn /unwarn /mute /unmute /ban /unban /kick /del`.
- 🔧 Снятие мута по таймеру (сейчас БД пишет `mute_until`, но авто-снятие
  полагается на Telegram `until_date` — добавить фоновую сверку).
- 🆕 Анти-рейд: всплеск входов → временный режим «только чтение» для новичков.
- 🆕 CAS-бан (база известных спам-ботов), проверка новичков.
- 🆕 Капча для новых участников (кнопка «я не бот») — опционально на чат.
- 🆕 Роль «модератор бота» (см. рефералы) с урезанными правами модерации.

## 2. Ранговая система

- ✅ E → D → C → B → A → S → SS → SSS по накопленному опыту (задания+подземелье).
- ✅ Карточка охотника `/rank`, прогресс-бар, авто-анонс повышения.
- ✅ Бонус Мана-руды при повышении ранга.
- ✅ **Привилегии рангов S/SS/SSS** (`/privileges`): надбавка к награде за
  задание и скидка комиссии перевода. Дневной лимит заданий одинаков для всех.
- 🆕 Сезоны/ладдер: сброс топа раз в месяц с наградами.

## 2.1 Задания-подписки (основной фарм + доход)

- ✅ Спонсорские каналы, награда рудой за подписку, проверка через бота-админа.
- ✅ Лесенка-стрик: множитель награды за сохранённые подписки.
- ✅ Ежедневная ре-проверка подписок (clawback) + мягкая блокировка без штрафов.
- ✅ **Дневной лимит** (по умолчанию 3/день, выше для S/SS/SSS).
- ✅ **Индивидуальный подбор без повторов** — каждому свой список на день,
  выполненные каналы не предлагаются снова (защита экономики).

## 3. Геймификация и удержание

- ✅ `/daily` — ежедневный бонус.
- ✅ `/top` — топ охотников, `/invites` — топ вербовщиков.
- 🆕 Достижения/бейджи (первое сообщение, 1000 сообщений, и т.д.).
- 🆕 Еженедельные челленджи с наградой в руде.

## 4. Реферальная система 🆕 (ключевая для роста)

### 4.1 Персональные ссылки
- `/invite` выдаёт **deep-link** вида `t.me/bot?start=ref_<userid>_<chatid>`.
- При входе нового пользователя по ссылке — фиксируем `referrals`, награждаем
  пригласившего рудой (`MANA_INVITE_BONUS`), исключаем самоприглашения и дубли.
- Поддержать также вступление в чат/канал по инвайт-ссылке (текущий механизм
  `on_new_member` уже ловит `from_user` — связать с deep-link).

### 4.2 Пороги повышения (задаёт админ чата)
- `/setgoal 5 moderator` — «пригласи 5 → роль модератор бота».
- `/setgoal 10 admin` — «пригласи 10 → предложить в админы» (фактическое
  Telegram-админство выдаёт владелец/создатель вручную — бот лишь уведомляет).
- `/setgoal 20 mana 1000` — «пригласи 20 → 1000 руды».
- Хранится в `referral_goals`, авто-проверка при каждом засчитанном реферале,
  авто-выдача роли (`chat_roles`) или награды + уведомление.

### 4.3 Реферальный кабинет
- `/myref` — моя ссылка, сколько привёл, прогресс до ближайшей цели.

## 5. Экономика — Мана-руда 🆕

- `/wallet` — баланс руды, всего намайнено/потрачено.
- `/mine` — забрать накопленную пассивную руду (если делаем пассивный фарм).
- `/shop` — магазин (см. MONETIZATION.md §3.2).
- `/transfer @user N` — перевод руды другому игроку.
- Заработок: сообщения, дейли, рефералы, ранги, топы (см. MONETIZATION.md §3.1).

## 6. Реклама 🆕

- `/newad` (владелец) — мастер создания рекламной кампании.
- `/ads` (владелец) — список кампаний и статистика.
- `/deletead` (владелец) — убрать неактуальную кампанию из активного списка.
- `/ads off|on` (админ чата) — управление рекламой в своём чате.
- Ежедневная авто-рассылка через планировщик, лимит 1/день на чат.
- Реклама в канале-витрине бота.
- (Будущее) рекламный кабинет для внешних рекламодателей с оплатой Stars.

## 7. Платежи 🆕

- Покупка пакетов руды за Telegram Stars.
- Оплата рекламных кампаний за Stars.
- Обмен руды на подарки/Premium (бот тратит свои Stars).
- `/donate` — поддержать проект (Stars), бонусом руда.
