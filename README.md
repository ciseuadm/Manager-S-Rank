## Возможности

### Автомодерация (24/7)
- Удаление **NSFW / 18+** контента
- Удаление **оскорблений** и мата
- Удаление **политического** контента
- Антиспам и антирекламный фильтр
- **Антифлуд** — детектирует и мутит за флуд
- Блокировка **Telegram инвайт-ссылок**
- Пользовательский **чёрный список слов**
- Опционально: фильтр ссылок, стикеров, CAPS LOCK

### Система наказаний
| Предупреждения | Действие |
|:-:|:-:|
| 1–2 | Предупреждение + удаление |
| 3+ | Автомут (60 мин по умолчанию) |
| 5+ | Автобан |

### Ранговая система (Solo Leveling)
| Ранг | Сообщений | Звание |
|:-:|:-:|:-:|
| ⬛ E-Ранг | 0+ | Новый охотник |
| 🟫 D-Ранг | 50+ | Начинающий охотник |
| 🟦 C-Ранг | 150+ | Опытный охотник |
| 🟩 B-Ранг | 400+ | Элитный охотник |
| 🟨 A-Ранг | 1000+ | Высший охотник |
| 🟥 S-Ранг | 2500+ | Монарх теней |
| 💎 SS-Ранг | 5000+ | Национальный охотник |
| 👑 SSS-Ранг | 10000+ | Monarchs Chosen |

### 🔹 Мана-руда (внутренняя валюта)
Глобальная валюта охотника (один кошелёк на все чаты). Добывается активностью.

| Источник | Награда (по умолчанию) |
|:--|:--:|
| Сообщение в чате | +1 (кулдаун 30 c) |
| Ежедневный бонус `/daily` | +50 |
| Приглашённый друг | +200 |
| Новый ранг | разовый бонус (100…25000) |

Команды: `/wallet`, `/shop`, `/transfer` (ответом), `/manatop`.
Пополнение за Telegram Stars: `/buy`. Поддержать проект: `/donate`.

### 🔗 Рефералы и VIP
- `/myref` — личная реф-ссылка (deep-link в бота).
- `/invite` — именная инвайт-ссылка в чат (засчитывает, кто кого привёл).
- Цели приглашений (админ): `/setgoal 5 moderator`, `/setgoal 10 admin`,
  `/setgoal 20 mana 1000` → авто-выдача роли/награды; список `/goals`.
- `/vip` — за **50+** приглашений в бота открывается VIP-зал с прямой связью
  с создателем (настраивается через `VIP_CHAT_LINK`).

### 📢 Реклама (монетизация)
- Владелец создаёт кампанию: `/newad` (мастер: контент → кнопка → дни).
- Планировщик рассылает активные кампании **раз в день** во все чаты,
  где бот админ. Жёсткий лимит — **1 рекламный пост в день на чат**.
- `/ads` — список и статистика, `/sendads` — разослать сейчас,
  `/pausead`, `/resumead`, `/deletead` — управление.
- Админ чата: `/ads on|off` — управление рекламой в своём чате.

> Обмен Мана-руды на подарки/Premium сейчас отвечает «Скоро появится...» —
> честный курс будет добавлен позже, чтобы не обесценить экономику.

## Установка

### 1. Клонируй репозиторий
```bash
git clone <your-repo>
cd "Manager S Rank"
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой .env
```bash
cp .env.example .env
```
Открой `.env` и вставь данные (полный список — в `.env.example`):
```env
BOT_TOKEN=1234567890:AAH...
OWNER_ID=123456789
LOG_CHANNEL_ID=-100123456789   # необязательно
VIP_CHAT_LINK=https://t.me/+xxxxx   # ссылка в VIP-зал (для 50+ рефералов)
BOT_CHANNEL_ID=-100...             # канал-витрина для рекламы (необязательно)
ADS_SEND_HOUR=12                   # час (UTC) ежедневной рекламной рассылки
```

### 4. Получи токен бота
Создай бота через [@BotFather](https://t.me/BotFather):
```
/newbot
```

### 5. Запусти бота
```bash
python bot.py
```

## Добавь бота в группу

1. Добавь бота в группу
2. Сделай его **администратором** со следующими правами:
   - ✅ Удалять сообщения
   - ✅ Блокировать пользователей
   - ✅ Ограничивать пользователей (мут)
3. Используй `/settings` для настройки фильтров

## Хостинг 24/7 (бесплатно)

### Вариант A — Railway (рекомендую, проще всего)
1. Залей проект на [GitHub](https://github.com) (приватный репозиторий).
2. Зарегистрируйся на [railway.app](https://railway.app) через GitHub.
3. **New Project → Deploy from GitHub repo** → выбери репозиторий.
4. Вкладка **Variables** → добавь переменные из `.env.example`:
   `BOT_TOKEN`, `OWNER_ID`, при желании `LOG_CHANNEL_ID`, `MODERATOR_IDS`.
5. Railway сам прочитает `Procfile`/`Dockerfile` и запустит бота 24/7.

### Вариант B — Fly.io (есть Docker)
1. Установи [flyctl](https://fly.io/docs/hands-on/install-flyctl/).
2. `fly launch` (использует `Dockerfile`), затем
   `fly secrets set BOT_TOKEN=... OWNER_ID=...`
3. `fly deploy`.

> ⚠️ База данных SQLite живёт в папке `data/`. На бесплатных хостингах файловая
> система может сбрасываться при передеплое — для постоянного хранения подключи
> volume (Railway Volume / Fly Volume) и смонтируй его в `/app/data`.
